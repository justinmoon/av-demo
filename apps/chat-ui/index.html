Usage

Open this page in two tabs with different roles, for example: 

?role=alice&relay=http://127.0.0.1:4443&nostr=ws://127.0.0.1:7447/&session=mydemo
?role=bob&relay=http://127.0.0.1:4443&nostr=ws://127.0.0.1:7447/&session=mydemo

The session parameter isolates BroadcastChannel handshakes. Make sure both tabs use the same session value.
